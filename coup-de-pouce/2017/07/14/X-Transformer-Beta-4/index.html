X-Transformer Beta 4 14 Jul 2017

Si vous n'avez pas encore lu mon précédent article sur Iridient X-Transformer, je vous invite à le faire, car je traiterai seulement des nouveautés apportées par la dernière version (Beta 4).

La nouveauté la plus attendue est certainement une intégration plus poussée du logiciel en tant que plug-in Lightroom, autant sur macOS que sur Windows. Plus besoin d'installer un plug-in tiers, il suffit donc d'aller dans le menu Help, puis de sélectionner Install Lightroom Plug-in.

Pour l'utiliser depuis Lightroom, vous pouvez sélectionner une ou plusieurs (enfin!) photos à partir du panneau Library, puis aller dans le menu File ou Library -> Plug-in Extras -> Process Selected RAW(s) to DNG

Quelques autres changements sont les bienvenus :

1) la possibilité de traduire le Film Simulation de Fujifilm en Camera Profile pour Lightroom, ce qui permet d'avoir les DNG résultants avec des couleurs très près des JPGs.

2) La possibilité de traduire les étoiles (Star Ratings) de la caméra, pour que Lightroom les applique.

3) X-Transformer peut maintenant conserver la date originale du fichier lors de la conversion RAW -> DNG.

On peut facilement voir l'effet que la conversion a sur les photos, les couleurs étant plus riches et nous avons plus de Dynamic Range.
